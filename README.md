# HMDB Spider

### HMDB Spider是专门用于爬取Human Metabolome Database（HMDB）中代谢物信息的程序，非常简单易用。不仅如此，本程序还非常贴心地将爬取的信息翻译成中文，有效缓解了非英语母语人士的痛苦。
